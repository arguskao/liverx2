PayPal綁定銀行或支票教學

September 11, 2017 arguskao

1 分鐘

 Twitter

 Facebook

 Email

PayPal提供綁定銀行或支票進行儲值的功能， 請先進入錢包連結至允許儲值的帳戶。(以下透過PayPal台灣地區示範，不同地區顯示畫面會有出入)

電子錢包連結 https://www.paypal.com/myaccount/wallet

支票請分別輸入銀行代碼與帳戶號碼

(如果不清楚請直接聯繫您所屬的銀行客服人員查詢)

若使用儲蓄帳戶請分別輸入銀行代碼與帳戶號碼

(如果不清楚請直接聯繫您所屬的銀行客服人員查詢)

請注意：

  1. 銀行帳戶與PayPal登記名稱必須相符，以利PayPal識別身分；

  2. 若您銀行帳戶為約定轉帳的模式，需事先將PayPal登記為收款人(查看收款人資訊)；

  3. 轉帳資金需要3~6天才能顯示於PayPal帳戶餘額中。

  4. 不同國家地區的PayPal連結銀行與儲值的規定不同，請參考常見問題或連繫客服。
