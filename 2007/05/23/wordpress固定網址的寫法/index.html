1 分鐘

最近讀到一篇相當不錯的文章，是三腳貓老師寫的，相當不錯，大家可以去參考。

WordPress 是 PHP 程式所建立的 Blog 系統。網頁 URL 的表達方式，原本不乏 filename.php?id=123 這類型式的寫法。不過，WordPress 的原作者群也都對於 SEO 深入的考慮，提供了固定網址（Permalinks）的檔名用法。

在此，不再討論固定鏈接（Permalinks）的檔名的重要性。主要在於討論如何設置？ 如何設置是最好？

固定網址如果要改成*.html 的方式，似乎會更好。
